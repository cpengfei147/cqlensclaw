# Governance Gate

Policy | Risk | Approval | Audit | Cost Budget

## Modules

- `policy-engine/` - Tool allowlist/blocklist, data access boundaries
- `risk-classifier/` - Action risk classification (low/medium/high/critical)
- `approval-engine/` - High-risk action approval workflow
- `audit-logger/` - Append-only operation logs
- `cost-budget/` - Token budget control
